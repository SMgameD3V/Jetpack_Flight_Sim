# 📖 Overview

**JetPack Flight Sim** is a gameplay prototype that combines traditional third-person movement with a custom jetpack flight system.

Built as my class thesis project, it demonstrates how an existing character controller can be extended with new gameplay mechanics while maintaining smooth transitions between walking and flying. The project focuses on movement programming, gameplay balancing, and interactive player feedback.
